Job Tracker Web App

Personal Project, April 2025

Tech: JavaScript, TypeScript, Node.js, Express, EJS, Passport.js, MongoDB, Mongoose, Prisma, PostgreSQL, Supabase, React

A full-stack web application that helps users track and manage their job applications. Originally built with JavaScript, Express, and MongoDB (Mongoose), then later refactored with TypeScript, Prisma, and PostgreSQL via Supabase to support scalability, maintainability, and typed backend development. Current version utilized a RESTful API backend with a React front-end.

Key Features

  • RESTful API for managing users and job applications
  • JWT-based authentication with rate limiting for security
  • PostgreSQL database with Prisma ORM for relational data modeling
  • Frontend forms with validation for creating and editing jobs
  • Jest test coverage for backend API endpoints
  • Tailwind CSS for responsive and modern UI styling
